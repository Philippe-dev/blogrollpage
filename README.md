# Pré-requis

Nécessite le plugin Template Helper.

## Informations générales

Blogroll Page vous permet d'afficher votre liste de lien (blogroll) sur une page dédiée, ce qui est particulièrement utile si vous avez des listes de liens à rallonge.

Vous pouvez utiliser le widget ou l'entrée prédéfinie de Simple Menu pour faire un lien vers cette page.
